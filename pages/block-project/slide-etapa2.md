# MCP e integração de ferramentas
#### **O MCP tem se popularizado como mecanismo de integração em agentes de IA**

<div class="h-5" />

::left::

<div class="w-full h-full flex flex-col justify-start gap-4 [&>*:first-child]:mt-0">

O **MCP (Model Context Protocol)** é um protocolo que padroniza como agentes de IA se conectam a ferramentas.

- Funciona como um **"USB"** para chatbot e agentes IA se conectarem a ferramentas externas.
- O MCP funciona através de um **servidor MCP** que expõe a ferramenta para o mundo externo (chatbot, Claude, IDEs, agentes de IA).

</div>

# Excalidraw para diagramas de arquitetura
#### **Excalidraw, draw.io e lucidchart são ótimas ferramentas para fazer diagramas**

<div class="h-5" />

::left::

<div class="w-full h-full flex flex-col justify-start gap-4 [&>*:first-child]:mt-0">

**Excalidraw**, **draw.io** e **Lucidchart** são ótimas ferramentas para desenhar diagramas de arquitetura.

- Todas oferecem licença **freemium**, algumas com **período trial** e **descontos para estudantes**.
- O **draw.io** (diagrams.net) pode ser usado **totalmente free** se instalado na máquina.

</div>
